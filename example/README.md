# Example

In order to run a demo app, run:

```
wasm-pack build --release
npm install
npm start
```

After that, go to [https://localhost:8080](https://localhost:8080) and open Console, you will see log messages. You can
try changing code in [src/lib.rs](https://github.com/wasm-rs/async-executor/blob/master/example/src/lib.rs) to see what it can do.
